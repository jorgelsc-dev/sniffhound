SniffHound

Native packet capture docs

SniffHound is a Python packet sniffer built with only native networking and threading primitives plus SQLite persistence. The runtime API and dashboard are provided through wsbuilder, while this site serves the public documentation landing page for sniffhound.jorgelsc.dev.

PyPI TestPyPI Source README

Capture model

The runtime stays intentionally small: raw socket capture, manual packet parsing, SQLite persistence, and UI-friendly summaries for sessions, packets, responses, tags, and statistics.

Usage

Install with pip install sniffhound and run sniffhound with elevated permissions when raw capture is unavailable. The web UI can switch into honeypot mode without extra command-line arguments.
